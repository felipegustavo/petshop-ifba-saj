# petshop-ifba-saj
Projeto de exemplo de um crud para a disciplina de Programação Web do IFBA SAJ

## Como rodar o projeto

```shell
mvn clean install spring-boot:run
```

## Como checar e aplicar o spotless

```shell
mvn spotless:check
```

```shell
mvn spotless:apply
```
